# diabetic-retinopathy-detection

Deep Learning project for Diabetic Retinopathy Detection using DenseNet121 and TensorFlow.

## Overview

This project focuses on detecting diabetic retinopathy from retinal fundus images using Deep Learning techniques.

## Technologies Used

- Python
- TensorFlow
- Keras
- NumPy
- Pandas
- OpenCV

## Model

- DenseNet121

## Dataset

- DDR (Diabetic Retinopathy Dataset)

## Learning Goals

- Learn Python
- Learn Machine Learning
- Learn Deep Learning
- Understand CNN architectures
- Understand medical image classification
